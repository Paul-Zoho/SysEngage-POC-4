What happens during Source Capture

  1. Pass 0 — Read Witness: File is read end-to-end; SHA-256 hash, byte count, and character count computed.
  2. Pass 0A — Segment Construction: Structural markers (headings, document boundaries) produce Segment groupings.
  3. Pass 0B — Source Capture: Paragraph-level verbatim Sources produced under LPM discipline (byte-for-byte preserved).
  4. Pass 0C — SourceAtom Splitting: Sources split into atomic units (sentence/line boundaries).

All entities committed atomically. AnalysisPass audit record produced per execution.
